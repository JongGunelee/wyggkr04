💡 대시보드 활용 가이드

이 대시보드를 활용하여 데이터를 기반의 의사결정을 내리는 최적의 분석 순서를 알아보세요.

🚀 최적 분석 순서 (Mind Map)

이 대시보드는 '전체 → 부분 → 상세' 순서로 문제를 파악하도록 설계되었습니다. 아래 단계를 따라 데이터 속 숨겨진 인사이트를 찾아보세요.

  1. STEP 1: 데이터 준비 및 설정 (⚙️)

    가장 먼저 정확한 분석을 위한 데이터를 준비하고 환경을 설정합니다.

  2. STEP 2: 전체 현황 파악 (숲 보기)

    상단의 '종합 개요'와 '핵심 요약'을 통해 '어디에', '무엇이' 문제인지 빠르게 파악합니다. 특히 데이터 품질 점수를 통해 데이터의 신뢰도를 먼저 확인하세요.

  3. STEP 3: 시나리오 기반 분석 (길 찾기)

    🚀 분석 시나리오 메뉴를 활용해 주요 분석 경로를 자동으로 탐색합니다. 필터와 화면이 시나리오에 맞게 최적화되어 복잡한 조작 없이 핵심 인사이트에 접근할 수 있습니다.

  4. STEP 4: 필터링과 시각적 탐색 (가지치기)

    직접 필터를 조정하며 가설을 세우고, 차트를 통해 '왜' 그런 현상이 나타나는지 추세와 패턴을 분석합니다. 차트 제목 옆 🔍 아이콘으로 '고급 필터'를 열어 특정 조건의 데이터만 심도있게 볼 수 있습니다.

  5. STEP 5: 심층 분석 및 검증 (뿌리 찾기)

    가장 강력한 계층 분석 기능으로 가설을 세우고 다각도로 데이터를 검증합니다. '작업 유형별 분석' 테이블에서 비교할 항목을 체크하고 [선택 항목 비교하기] 버튼을 클릭하세요.

  6. STEP 6: 결과 저장 및 공유 (보고)

    분석 결과를 원본 그대로 저장하거나 데이터로 내보내 공유합니다.

STEP 1: 데이터 준비 및 설정 (⚙️)

Q: 분석을 시작하기 전에 무엇을 해야 하나요?

A: 정확한 분석을 위해 우측 상단 ⚙️ (데이터 및 설정) 메뉴에서 아래 항목들을 확인하고 준비합니다.

  1. 데이터 업로드: 엑셀/CSV 파일 업로드를 통해 최신 데이터를 불러옵니다. (필요 시 양식 다운로드 활용)
  2. 세대수 정보 확인: 동별 세대수 관리에서 각 동의 전체 세대수 정보가 정확한지 확인합니다. 이 정보는 '진척률' 계산의 기준이 됩니다.
  3. (선택) 화면 레이아웃 변경: 와이드 모니터를 사용하신다면 화면 표시 > 레이아웃 변경에서 '왼쪽' 또는 '오른쪽' 사이드바를 선택하여 더 넓은 차트 영역을 확보할 수 있습니다.
STEP 2: 전체 현황 파악 (숲 보기)

💯 데이터 품질 점수 이해하기

'종합 개요'의 데이터 품질 카드는 현재 데이터의 신뢰도를 100점 만점으로 보여줍니다. 점수가 낮으면 분석 결과가 왜곡될 수 있습니다.

  • 점수 하락 요인: 필수 항목(PO번호, 연도 등) 누락, PO금액 0원, 동별 전체 세대수 정보 누락 등.
  • 점수 향상 방법:
    1. 카드 우측 상단의 ℹ️ 아이콘에 마우스를 올려 품질 저하 원인을 상세히 확인합니다.
    2. 원본 엑셀/CSV 파일에서 해당 데이터를 수정합니다.
    3. 수정된 파일을 다시 업로드합니다. (세대수 정보는 동별 세대수 관리 메뉴에서 직접 수정 가능)
STEP 3: 시나리오 기반 분석 (길 찾기)

🚀 분석 시나리오 활용하기

필터 헤더의 🚀 분석 시나리오 버튼은 복잡한 필터 조작 없이, 클릭 한 번으로 특정 목적에 맞는 분석 환경을 자동으로 구성해주는 강력한 기능입니다.

  • 최근 3개년 비용 집중도 분석: 자동으로 최근 3개년 데이터를 필터링하고, 비용 분석에 필요한 모든 차트와 테이블이 포함된 아코디언을 펼쳐줍니다.
  • 비용 Top3 작업유형 연간 추이: 현재 데이터에서 비용이 가장 많이 발생한 상위 3개 작업 유형을 자동으로 찾아, 해당 유형들의 연간 비용 추이 차트를 보여줍니다.
  • 견적 초과 발생 현황 분석: 모든 작업 유형을 대상으로 '계층 분석' 모달을 실행하고, 견적을 초과한 항목을 분석하기 위한 최적의 설정(분석 모델, 필터 등)을 자동으로 적용해줍니다.
STEP 4: 필터링과 시각적 탐색 (가지치기)

필터링된 데이터를 아래 차트들을 통해 시각적으로 탐색하며 '왜' 이런 결과가 나왔는지 원인을 추적합니다.

질문별 추천 차트:
  • "특정 작업(예: 씽크대 교체) 비용이 왜 유독 A동에서만 높을까?"
    → 동별 총 PO금액 및 세대당 평균금액 차트를 확인하세요. '총 PO금액'(막대)은 높은데 '세대당 평균금액'(선)이 낮다면, 작업 건수가 많기 때문입니다. 반대의 경우, 소수의 고비용 작업이 있었음을 의미합니다.
  • "특정 작업 유형 또는 특정 동의 연간 비용/건수 추세가 궁금하다면?"
    → 작업 유형별 연간 추이 차트의 🔍 아이콘을 클릭하여 '고급 필터'를 여세요. 원하는 작업 유형, 동, 연도, 분석 지표(비용/건수)를 자유롭게 조합하여 맞춤형 추이 분석이 가능합니다.
  • "A동과 B동의 특정 작업 유형에 대한 연도별 실적을 직접 비교하고 싶다면?"
    → 동-작업유형별 연간 비용 및 건수 비교 차트의 🔍 아이콘을 클릭하여 '고급 필터'를 여세요. 비교하고 싶은 동, 작업 유형, 연도를 직접 선택하여 상세 비교 분석을 수행할 수 있습니다.
STEP 5: 심층 분석 및 검증 (뿌리 찾기)

🚀 가장 강력한 기능: 선택 항목 비교하기 (계층 분석)

엑셀의 '피벗 테이블'처럼, 여러 기준으로 데이터를 동적으로 그룹화하고 재정렬하여 다각도로 심층 분석하는 기능입니다. 가설을 세우고 최종적으로 검증하는 단계에서 사용합니다.

활용 시나리오: "견적을 초과한 작업들은 주로 특정 건물과 특정 작업 유형에 몰려있을 것이다."

  1. 작업 유형별 분석 테이블에서 분석하고 싶은 작업 유형들을 모두 체크합니다.
  2. 우측 상단의 [선택 항목 비교하기] 버튼을 클릭합니다.
  3. 분석 컨트롤 메뉴를 아래와 같이 설정합니다. (메뉴가 접혀있다면 ▼ 버튼으로 펼칠 수 있습니다.)
    • 분석 모델: '견적 비교 > 건물동 > 작업유형' 순으로 선택하여 데이터를 그룹화합니다.
    • 비교 연도/지표: 분석할 연도와 'PO금액', '작업 건수'를 선택합니다.
    • 기본견적 금액: 연도별 기준 견적 금액을 입력합니다.
    • 견적 비교 필터: '초과'를 선택합니다.
  4. 결과 확인: 이제 테이블에는 '견적을 초과한' 작업들만 '건물동'과 '작업유형'별로 그룹화되어 표시됩니다. 이를 통해 어떤 건물, 어떤 작업에서 비용 초과가 빈번했는지 명확하게 확인할 수 있습니다.
  5. 추가 기능:
    • 계층 보기: [모두 접기], [1단계] 등 버튼으로 결과 테이블의 가독성을 높일 수 있습니다.
    • 전체 화면: [전체 화면] 버튼으로 결과를 더 넓게 보고, 🎚️ 버튼으로 필터 패널을 열어 실시간으로 조건을 변경하며 분석할 수 있습니다.
    • 내보내기: [엑셀로 내보내기] 버튼으로 현재 보이는 그대로의 분석 결과를 파일로 저장할 수 있습니다.
STEP 6: 결과 저장 및 공유 (보고)

Q: 분석한 이 화면을 그대로 저장하고 싶어요?

A: ⚙️ (데이터 및 설정) 메뉴의 현재 화면(HTML) 저장 버튼을 클릭하세요.

이 기능은 현재 데이터, 모든 필터 값, 차트의 범례 활성화 상태, 심지어 화면 레이아웃과 크기까지 모든 것을 포함한 '완벽한 스냅샷'을 하나의 HTML 파일로 저장합니다.

저장된 파일을 나중에 열면, 지금 보고 있는 이 분석 화면 그대로 완벽하게 복원됩니다. 팀원과의 공유나 개인적인 분석 기록 보관에 매우 유용합니다.
