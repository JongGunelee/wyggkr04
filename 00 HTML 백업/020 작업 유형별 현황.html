이 대시보드를 활용하여 데이터 기반의 의사결정을 내리는 방법을 알아보세요.

0. 데이터 필터 활용법

Q: 데이터 필터는 어떻게 사용하나요?

A: 상단의 데이터 필터 영역에서 원하는 '연도', '유형', '건물' 등을 조합하여 데이터를 상세 분석할 수 있습니다. 필터링된 결과는 아래 모든 차트와 요약 카드에 실시간으로 반영됩니다.

Q: 필터의 각 버튼은 어떤 기능인가요?

  • 🔎: 필터 항목이 많을 때, 팝업창에서 검색하여 원하는 항목을 쉽게 찾고 선택할 수 있습니다.
  • ➕ / ➖: 모든 항목을 한 번에 선택하거나 선택 해제합니다.
  • ▲ / ▼: 목록을 오름차순 또는 내림차순으로 정렬합니다.
  • 🔝: 현재 선택한 항목들을 목록의 최상단으로 고정합니다. 자주 보는 항목들을 고정시켜두면 편리합니다. (한 번 더 누르면 해제)

Q: 필터 활용 예시를 보여주세요.

예시: '23년'과 '24년'에 '8동'과 '9동'에서 진행된 '씽크대 교체' 작업만 보고 싶을 때

  1. '연도' 필터에서 '23년', '24년'을 선택합니다 (Ctrl 또는 Shift 키를 누른 채 클릭).
  2. '유형' 필터에서 '사택 씽크대 교체'를 선택합니다.
  3. '건물' 필터에서 '8동', '9동'을 선택합니다.
  4. 이제 모든 차트와 표는 해당 조건에 맞는 데이터만으로 재구성됩니다.
1. 주요 지표(KPI) 이해하기

Q: '종합 개요' 카드의 값들은 무엇을 의미하나요?

A: 각 카드는 현재 필터링된 데이터를 기준으로 계산된 핵심 성과 지표(KPI)를 보여줍니다.

  • 작업 건수: 현재 필터 조건에 맞는 데이터의 총 개수(행의 수)입니다.
  • 집행 금액: 필터링된 데이터들의 PO금액을 모두 합산한 총액입니다.
  • 건당 평균 비용: '총 집행 금액'을 '총 작업 건수'로 나눈 값입니다. 즉, (총 집행 금액) / (총 작업 건수) 로 계산됩니다.

Q: YoY가 무엇인가요?

A: YoY는 Year-on-Year의 약자로, '전년 대비 증감률'을 의미합니다. 금년 실적을 작년 실적과 비교하여 추세를 파악하는 데 사용됩니다.
((금년 실적 - 전년 실적) / 전년 실적) * 100%로 계산되며, 아이콘으로 ▲ 증가, ▼ 감소를 쉽게 알 수 있습니다.

2. 핵심 인사이트로 문제점 빠르게 파악하기

Q: 가장 먼저 무엇을 봐야 하나요?

A: 핵심 인사이트 요약 섹션의 카드들을 먼저 확인하세요. 이 카드들은 현재 필터 조건 내에서 '어디에 돈이 많이 쓰이는지', '어디서 작업이 자주 발생하는지'를 빠르게 알려주어 심층 분석의 시작점을 제공합니다.

  • 비용 발생 상위 동: 어떤 건물에 가장 많은 예산이 투입되었는지 보여줍니다.
  • 작업 빈도 상위 동: 어떤 건물에서 문제가 가장 자주 발생했는지 보여줍니다.
  • 비용 발생 상위 작업 유형: 어떤 작업에 가장 많은 비용이 지출되었는지 보여줍니다.

이 카드들의 항목을 클릭하면 해당 데이터의 상세 내역을 바로 확인할 수 있습니다.

3. 차트와 테이블로 심층 분석하기

아래의 차트들은 필터 조건에 따라 동적으로 변경되며, 데이터의 추세와 특징을 시각적으로 분석하는 데 도움을 줍니다.

동별 분석 차트 가이드

1. 동-작업유형별 연간 비용 및 건수 비교
  • 목적 및 인사이트:
    특정 동과 특정 작업의 조합이 연도별로 어떻게 변화했는지 비용(막대)과 건수(선)를 동시에 추적합니다. '8동의 씽크대 교체 비용이 작년에 왜 급증했는지'와 같은 구체적인 질문에 대한 답을 찾는 데 가장 효과적인 차트입니다.
  • 사용 방법:
    1. 데이터 필터에서 분석하고 싶은 `건물`과 `유형`을 하나 이상 선택합니다.
    2. 차트 상단 범례의 **작업 유형명 앞 체크박스**를 사용하면, 해당 유형에 속한 모든 동의 데이터를 한 번에 켜고 끌 수 있어 특정 항목에 집중하기 용이합니다.
    3. 범례의 색상 심볼(네모, 선)을 클릭하여 특정 동의 데이터를 개별적으로 숨기거나 보일 수 있습니다.
2. 동별 총 PO금액 및 세대당 평균금액
  • 목적 및 인사이트:
    특정 동에 비용이 많이 드는 '이유'를 파악합니다. 총비용(막대)은 높은데 세대당 평균비용(선)이 낮다면, 작업 단가보다는 세대수가 많거나 작업 건수가 많기 때문입니다. 반대로, 총비용은 낮아도 세대당 평균비용이 높다면 소수의 고비용 작업이 있었음을 의미합니다.
  • 사용 방법:
    필터링된 결과에 따라 모든 동의 재무 건전성을 한눈에 비교할 수 있습니다. 특정 동의 막대나 선 위의 점에 마우스를 올리면 정확한 값을 볼 수 있습니다.
3. 동별 집행비 및 작업 건수
  • 목적 및 인사이트:
    필터 조건 내에서 각 동의 총비용(막대)과 총 작업 건수(선)를 직관적으로 비교합니다. 어떤 동에서 작업이 가장 빈번하고, 비용이 가장 많이 발생했는지 빠르게 파악할 수 있습니다. `핵심 요약` 카드의 '상위 동' 정보와 함께 보면 유용합니다.
4. 동 및 작업 유형별 진척률
  • 목적 및 인사이트:
    세대수가 정의된 대규모 프로젝트의 진행 현황을 관리합니다. 연도별 작업 건수(막대), 누적 건수(선), 그리고 누적 진척률(%)을 통해 목표 대비 실적을 추적할 수 있습니다.
  • 사용 방법:
    1. 필터에서 진척률을 보고 싶은 `유형`과 `건물`을 선택합니다.
    2. ⚙️(설정) 메뉴의 `동별 세대수 관리`에서 해당 동의 '전체 세대수'가 정확히 입력되었는지 확인해야 정확한 진척률이 계산됩니다.

작업 유형별 분석 차트 가이드

1. 작업 유형별 건수 및 총 PO 금액
  • 목적 및 인사이트:
    어떤 작업 유형에 가장 많은 예산과 빈도가 집중되는지 파악합니다. 예산 배분의 우선순위를 정하거나 특정 작업의 비용 구조를 검토하는 데 기초 자료로 활용됩니다.
2. 작업 유형별 연간 비용 추이
  • 목적 및 인사이트:
    선택한 작업 유형들의 연도별 비용 추이를 한눈에 비교합니다. 특정 작업의 비용이 급증하거나 감소하는 추세를 파악하여 예산 계획 수립이나 이상 비용 감지에 활용할 수 있습니다.
  • 사용 방법:
    필터에서 분석하고 싶은 `유형`을 하나 이상 선택하면 해당 유형들의 연간 비용 추이가 꺾은선 그래프로 표시됩니다.

강력 추천: 선택 항목 비교하기 (계층 분석)

  • 목적 및 인사이트:
    엑셀의 '피벗 테이블'처럼, 여러 기준으로 데이터를 동적으로 그룹화하고 재정렬하여 다각도로 심층 분석하는 **가장 강력한 기능**입니다. 원본 데이터(Raw Data)를 직접 보지 않고도 원하는 관점의 요약 테이블을 즉시 생성할 수 있습니다.
  • 사용 방법:
    1. `작업 유형별 분석` 테이블에서 비교하고 싶은 작업 유형들을 **체크박스로 선택**합니다.
    2. 우측 상단의 **[선택 항목 비교하기]** 버튼을 클릭하여 분석 모달창을 엽니다.
    3. 분석 컨트롤 메뉴**를 활용하여 원하는 리포트를 만듭니다.
      • 분석 모델: 데이터 그룹화 순서를 결정합니다. **가장 중요한 설정**으로, 예를 들어 `건물동 > 작업유형`을 선택하면 데이터를 건물동으로 먼저 묶고, 그 안에서 작업유형별로 다시 묶어 보여줍니다.
      • 비교 연도/지표: 분석할 연도와 테이블의 열(Column)에 표시될 데이터 항목(PO금액, 건수 등)을 선택합니다.
      • 기본견적 금액: 특정 연도의 견적 금액을 입력하고 `견적 비교` 필터를 '초과' 등으로 설정하면, 견적 대비 실제 집행액이 초과된 항목만 필터링하여 볼 수 있습니다.
  • 결과 활용:
    • 생성된 테이블의 그룹 헤더(회색 배경 행)를 클릭하여 하위 항목을 접거나 펼 수 있습니다.
    • 테이블 우측 상단의 **[전체 화면]** 버튼을 클릭하면 더 넓은 화면에서 데이터를 볼 수 있으며, 이때 나타나는 **플로팅 필터 패널**을 자유롭게 이동/크기 조절하며 더욱 편리하게 분석할 수 있습니다.
    • **[엑셀로 내보내기]** 버튼으로 현재 분석된 테이블을 바로 다운로드할 수 있습니다.
4. 데이터 관리 및 개인화

Q: 최신 데이터를 반영하려면 어떻게 하나요?

A: 오른쪽 상단 ⚙️ 아이콘을 클릭하여 데이터 관리 메뉴를 여세요. 엑셀/CSV 파일 업로드를 통해 새로운 데이터를 불러올 수 있습니다. 엑셀/CSV 양식 다운로드 버튼으로 필요한 데이터 형식을 다운로드할 수 있습니다.

Q: 나만의 화면 배치를 사용하고 싶어요?

A: 데이터 관리 메뉴의 레이아웃 변경 옵션을 사용해 보세요. 필터 영역을 기존의 '상단' 방식 외에 '왼쪽' 또는 '오른쪽' 사이드바 형태로 변경할 수 있습니다. 와이드 스크린 사용자에게 특히 유용합니다.

Q: 현재 분석 화면을 저장하고 싶어요?

A: 데이터 관리 메뉴의 HTML 다운로드 버튼을 클릭하세요. 현재 데이터와 필터 설정, **그리고 선택한 레이아웃까지** 모두 포함된 하나의 HTML 파일로 저장되어, 언제든지 이 파일을 열어 지금의 분석 상태를 그대로 다시 볼 수 있습니다.
