💡 대시보드 활용 가이드

이 대시보드를 활용하여 데이터를 기반의 의사결정을 내리는 최적의 분석 순서를 알아보세요.

🚀 최적 분석 순서 (Mind Map)

이 대시보드는 '전체 현황 파악 → 가설 수립 → 심층 분석 및 검증' 순서로 문제를 파악하도록 설계되었습니다. 아래 단계를 따라 데이터 속 숨겨진 인사이트를 찾아보세요.

  1. STEP 1: 데이터 준비 및 설정 (⚙️)

    가장 먼저 정확한 분석을 위한 데이터를 준비합니다. 데이터 품질 점수를 확인하여 데이터의 신뢰도를 점검하고, 필요 시 화면 레이아웃을 변경하여 분석 환경을 최적화합니다.

  2. STEP 2: 전체 현황 파악 (숲 보기)

    상단의 '종합 개요', '핵심 요약', '핵심 차이' 카드를 통해 '어디에', '무엇이' 문제인지 빠르게 파악하고 분석의 방향을 설정합니다.

  3. STEP 3: 필터링 마스터하기 (핵심 기능)

    가설을 검증하기 위해 다양한 필터 기능을 활용하여 데이터를 탐색합니다. 이 대시보드의 가장 핵심적인 기능으로, 수동/자동/고급 필터를 조합하여 사용합니다.

  4. STEP 4: 시나리오 기반 분석 (길 찾기)

    🚀 분석 시나리오 메뉴를 활용해 주요 분석 경로를 자동으로 탐색합니다. 클릭 한 번으로 복잡한 필터 조작 없이 특정 목적에 맞는 분석 환경을 구성할 수 있습니다.

  5. STEP 5: 심층 분석 및 검증 (뿌리 찾기)

    가장 강력한 계층 분석 기능으로 가설을 세우고 다각도로 데이터를 검증합니다. '작업 유형별 분석' 테이블에서 비교할 항목을 체크하고 [선택 항목 비교하기] 버튼을 클릭하여 피벗 테이블 형태의 상세 분석을 수행하세요.

  6. STEP 6: 결과 저장 및 공유 (보고)

    분석이 완료된 화면을 [현재 화면(HTML) 저장] 기능으로 원본 그대로 저장하거나, 필요한 데이터를 엑셀 파일로 내보내 보고 자료로 활용합니다.

STEP 1: 데이터 준비 및 설정 (⚙️)

정확한 분석을 위해 우측 상단 ⚙️ (데이터 및 설정) 메뉴에서 아래 항목들을 확인하고 준비하세요.

  1. 데이터 업로드: 엑셀/CSV 파일 업로드를 통해 최신 데이터를 불러옵니다. (필요 시 양식 다운로드 활용)
  2. 세대수 정보 확인: 동별 세대수 관리에서 각 동의 전체 세대수 정보가 정확한지 확인합니다. 이 정보는 '진척률' 및 '세대당 평균금액' 계산의 기준이 됩니다.
  3. (선택) 화면 레이아웃 변경: 와이드 모니터를 사용하신다면 화면 표시 > 레이아웃 변경에서 '왼쪽' 또는 '오른쪽' 사이드바를 선택하여 더 넓은 차트 영역을 확보할 수 있습니다. 사이드바는 핀(📌) 아이콘으로 고정하거나 숨길 수 있어, 필요에 따라 작업 공간을 유연하게 조절할 수 있습니다.

(관리자용) 중앙 관리 설정 변수란?

이 대시보드는 특정 비즈니스 규칙을 코드 상단의 '중앙 관리 설정 변수' 구역에서 통합 관리합니다. 규칙 변경이 필요할 경우, 코드의 해당 부분만 수정하면 시스템 전체에 안전하게 반영됩니다.

  • EXCLUDED_ORDERERS: 분석에서 기본적으로 제외할 '발주처' 목록입니다.
  • DEPARTMENT_PREFIX: 제외할 '부서' 이름의 시작 패턴입니다.
  • LEASE_WORK_TYPE: '외부임차' 그룹을 식별하는 데 사용되는 '작업유형'의 정확한 이름입니다.

참고: 이 설정들은 코드에 직접 접근해야 변경할 수 있으므로, 대시보드 관리자가 담당합니다.

STEP 2: 전체 현황 파악 (숲 보기)

💯 데이터 품질 점수란 무엇인가요?

'종합 개요'의 데이터 품질 카드는 현재 데이터의 신뢰도를 100점 만점으로 보여줍니다. 점수가 낮으면 분석 결과가 왜곡될 수 있으므로, 분석 시작 전 반드시 확인하세요.

  • 점수 하락 요인: 필수 항목(PO번호, 연도 등) 누락, PO금액 0원, 동별 전체 세대수 정보 누락 등.
  • 점수 향상 방법:
    1. 카드 우측 상단의 ℹ️ 아이콘에 마우스를 올려 품질 저하 원인을 상세히 확인합니다.
    2. 원본 엑셀/CSV 파일에서 해당 데이터를 수정합니다.
    3. 수정된 파일을 다시 업로드합니다. (세대수 정보는 동별 세대수 관리 메뉴에서 직접 수정 가능)

👀 핵심 요약 및 핵심 차이 카드는 어떻게 활용하나요?

'핵심 요약' 섹션은 필터링된 데이터를 기반으로 비용 및 빈도 기준 상위 3개 항목을 보여주어 '어디에' 자원이 집중되고, '어떤 작업이' 자주 발생하는지 직관적으로 파악할 수 있습니다. '핵심 차이' 섹션은 전년 동기 대비 비용 증감 및 신규/급증 항목을 보여주어 변화의 추세를 빠르게 감지하도록 돕습니다. 각 항목을 클릭하면 관련 상세 데이터 팝업이 나타납니다.

STEP 3: 필터링 마스터하기 (핵심 기능)

기본 필터 (수동)

필터 목록에서 원하는 항목을 직접 선택하여 데이터를 탐색합니다.

  • 다중 선택: Ctrl 키를 누른 채 클릭하여 여러 항목을 선택하거나, Shift 키로 범위를 선택할 수 있습니다.
  • 임시 저장: [임시저장] 버튼은 현재 필터 상태를 '초기 상태'로 저장합니다. 이후 [필터 초기화] 버튼을 누르면 이 상태로 돌아옵니다.
  • 필터 컨트롤 아이콘: 필터 목록 우측의 아이콘들을 활용하여 필터링 효율을 높일 수 있습니다.
    • 🔎 (검색): 항목이 많을 때 클릭하여 검색 및 다중 선택 창을 엽니다.
    • ✓ (전체 선택/해제): 목록의 모든 항목을 한 번에 선택하거나 해제합니다.
    • ▲▼ (정렬): 목록의 정렬 순서를 오름차순/내림차순으로 변경합니다.
    • 🔝 (상단 고정): 현재 선택한 항목들을 목록의 최상단으로 고정시켜 쉽게 접근할 수 있습니다. (다시 누르면 해제)

자동 필터 (자동)

🚀 분석 시나리오나 📅 연도 필터를 사용하면 관련 필터들이 자동으로 최적화됩니다. 이 상태에서는 데이터의 정합성을 위해 수동 필터 조작이 잠깁니다. 수동으로 다시 필터를 조작하려면 [필터 초기화] 버튼을 눌러 자동 필터 상태를 해제하세요.

고급 필터 (차트별)

'동-작업유형별 비교', '작업 유형별 연간 추이' 등 일부 차트 제목 옆의 🔍 아이콘을 클릭하면 해당 차트에만 적용되는 고급 필터 창이 열립니다. 이를 통해 전체 필터에 영향을 주지 않고 특정 차트의 데이터만 세밀하게 분석할 수 있습니다. 고급 필터가 적용된 차트는 아이콘이 붉은색(🔴)으로 표시됩니다.

⚠️ 중요: 자동 제외 규칙 안내

데이터의 일관성과 분석 정확도를 위해, 특정 조건의 데이터는 모든 분석에서 기본적으로 제외됩니다. 이 규칙은 (관리자용) 중앙 관리 설정 변수를 통해 관리됩니다.

  • 제외 규칙: 발주처 '엘지화학', 'u' 및 '0연구부서 기획_'으로 시작하는 부서.
  • 적용 시점: 필터 초기화, 데이터 로딩, 시나리오 적용 시 항상 자동으로 반영됩니다. (사용자가 필터에서 직접 해당 항목을 선택하여 일시적으로 포함시킬 수는 있습니다.)
STEP 4: 시나리오 기반 분석 (길 찾기)

🚀 분석 시나리오 활용하기

필터 헤더의 🚀 분석 시나리오 버튼은 복잡한 필터 조작 없이, 클릭 한 번으로 특정 목적에 맞는 분석 환경을 자동으로 구성해주는 강력한 기능입니다. 주요 분석 경로를 빠르게 탐색할 수 있습니다.

  • 금년도 집행비: 최신 연도에 비용이 집중된 동과 작업 유형을 확인합니다.
  • 월간 집행비(단독+공용/공용): 최근 2개년 간의 월별 비용 투입 현황을 '계층 분석' 모달에서 확인하기 위한 최적의 설정을 자동으로 적용합니다.
  • 집행비 상승 Top3: 최신 연도 기준, 비용이 가장 많이 발생한 상위 3개 작업 유형을 자동으로 필터링하여 심층 분석을 준비합니다.
STEP 5: 심층 분석 및 검증 (뿌리 찾기)

🚀 핵심 기능: 선택 항목 비교하기 (계층 분석)

엑셀의 '피벗 테이블'처럼, 여러 기준으로 데이터를 동적으로 그룹화하고 재정렬하여 다각도로 심층 분석하는 기능입니다. 가설을 세우고 최종적으로 검증하는 단계에서 사용하세요.

활용 예시: "견적을 초과한 작업은 주로 어떤 건물과 작업 유형에 몰려있을까?"

  1. 작업 유형별 분석 테이블에서 분석하고 싶은 작업 유형들을 모두 체크합니다.
  2. 우측 상단의 [선택 항목 비교하기] 버튼을 클릭합니다.
  3. 분석 컨트롤 메뉴를 아래와 같이 설정합니다. (메뉴가 접혀있다면 ▼ 버튼으로 펼칠 수 있습니다.)
    • 분석 모델: '견적 비교 > 건물동 > 유형' 또는 '발주처 > 계약명' 등 다양한 모델을 선택하여 데이터를 그룹화합니다.
    • 비교 연도/지표: 분석할 연도와 'PO금액', '작업 건수'를 선택합니다.
    • 동별 기본 금액: 그룹별로 기준 견적 금액을 입력합니다. '외부임차' 그룹에 입력된 금액은 '임차사택 공가 수장공사' 작업에만 정밀하게 적용됩니다.
    • 견적 비교 필터: '초과'를 선택합니다.
  4. 결과 확인 및 추가 기능:
    • 테이블 상호작용:
      • 정렬: 컬럼 헤더를 클릭하면 그룹 합계 기준, Shift+클릭하면 개별 항목 기준으로 정렬됩니다. 계층 헤더의 △ 아이콘을 클릭하면 시나리오 기본 정렬과 가나다순 정렬 간에 전환할 수 있습니다.
      • 계층 보기: [모두 접기], [1단계] 등 버튼으로 결과 테이블의 가독성을 높이거나, 각 그룹 헤더를 직접 클릭하여 개별적으로 접고 펼 수 있습니다.
      • 전체 화면 및 크기 조절: [전체 화면] 버튼으로 결과를 더 넓게 보고, 테이블의 행/열 경계선을 드래그하여 크기를 자유롭게 조절할 수 있습니다.
    • 내보내기: [엑셀로 내보내기] 버튼으로 현재 보이는 그대로의 분석 결과를 파일로 저장합니다.
STEP 6: 결과 저장 및 공유 (보고)

Q: 분석한 이 화면을 그대로 저장하고 싶어요.

A: ⚙️ (데이터 및 설정) 메뉴의 [현재 화면(HTML) 저장] 버튼을 클릭하세요.

이 기능은 현재 데이터, 모든 필터 값, 차트 범례 상태, 화면 레이아웃, 심지어 계층 분석 테이블의 정렬 및 확장 상태까지 모든 것을 포함한 '완벽한 스냅샷'을 하나의 HTML 파일로 저장합니다. 저장된 파일을 나중에 열면, 지금 보고 있는 이 분석 화면 그대로 완벽하게 복원됩니다.

팀원과의 공유나 개인적인 분석 기록 보관에 매우 유용합니다.
