🔬 작동 원리
북마크릿은 다음과 같은 방식으로 메인 HTML을 수정하지 않고 성능을 측정합니다: 
1. 북마크릿 클릭 → JavaScript 코드가 현재 페이지에 주입됨
2. window 객체의 함수를 Proxy로 래핑하여 호출 추적
   예: querySelectorAll, DataManager.getProjects 등
3. Performance API로 각 작업의 시작/종료 시간 측정
4. Mutation Observer로 DOM 변경 사항 감지
5. 측정 데이터를 우측 하단 플로팅 패널에 실시간 표시
6. 세션 종료 시 CSV 파일로 다운로드 가능

핵심 장점: 메인 HTML 파일을 전혀 수정하지 않으므로 Section 0-START 규칙 #4 (진단 코드 주입 금지)를 완벽히 준수합니다.
